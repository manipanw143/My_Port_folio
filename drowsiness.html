Description

This project is a comprehensive solution for Community Business and Emeelan Platform. I contributed as a core team member to enhance features and improve performance for this platform.

Features

  • Built a custom REST API that returns counts of Business Types for client-side tables, verified with Postman.
  • Optimized the Dashboard component to reduce landing page loading times and implemented pagination for all tables to enhance data retrieval efficiency based on page size and current page.
  • Created a service to import Pincode data into a PostgreSQL collection for streamlined access.
  • Cleaned and imported data from Excel to PostgreSQL for testing purposes.

Tools & Technology

  • HTML, CSS
  • React.js, Ant Design
  • Strapi (CMS)
  • PostgreSQL
  • Docker, Postman
