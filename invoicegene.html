Invoice Generator | September 2023 - Present

Bangalore, Karnataka

Developed a comprehensive invoice generator web application using React.js with Ant Design for UI components. Backend services were created using Strapi, enabling seamless data management and customization with Refine.

Key Features:

  • CRUD Operations: Implemented backend API services to manage Create, Read, Update, and Delete operations for invoice data.
  • Dynamic Data Population: Built a feature that auto-populates the Producttype collection with combinations of Product and Type entries.
  • Invoice Search Filter: Implemented a search filter that allows users to quickly retrieve invoices by InvoiceNo.
  • Customizable Data Management: Backend was built using Strapi, enabling easy customization and content management of invoices and related data.

Tools & Technology:

  • Frontend: HTML, CSS, TypeScript, React.js, Ant Design
  • Backend: Strapi (CMS), Refine
  • Database: Designed schema for efficient data storage with tables/collections such as Product, Type, Producttype, Tax, Invoice, InvoiceItems, Purity, and Customer.

Impact:

  • Efficient Data Management: The system enables seamless management of invoice data with a scalable backend powered by Strapi.
  • Improved User Experience: Ant Design provides a sleek and intuitive interface, improving usability for users managing invoices.
  • Enhanced Searchability: The Invoice Search Filter makes it easier for users to find specific invoices quickly, saving time and increasing productivity.
